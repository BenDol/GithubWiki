# Guides

Welcome to the guides section! Here you'll find detailed tutorials and how-to guides for working with the GitHub Wiki Framework.

## Available Guides

### Content Management
- **Creating Pages** - Learn how to create and organize wiki pages
- **Editing Pages** - Edit existing pages and submit changes via pull requests
- **Organizing Content** - Best practices for structuring your wiki

### Customization
- **Theming** - Customize colors, fonts, and styles
- **Configuration** - Configure sections, features, and settings
- **Adding Sections** - Create new sections for your wiki

### Advanced Topics
- **Search Setup** - Configure and optimize search functionality
- **GitHub Integration** - Set up OAuth and repository integration
- **Deployment** - Deploy your wiki to GitHub Pages or other platforms

## Getting Help

If you need help with something not covered in these guides:

1. Check the [Reference](../reference/index) section for technical details
2. Search the wiki for related content
3. Open an issue on GitHub for questions or bug reports
4. Contribute by writing new guides!

## Contributing Guides

Want to contribute a guide? Follow these steps:

1. Create a new markdown file in `public/content/guides/`
2. Add appropriate frontmatter
3. Write clear, step-by-step instructions
4. Include code examples and screenshots
5. Submit a pull request
